## Get started

1. Install dependencies

   ```bash
   git clone https://github.com/m0bilemike/tryhackmeApp
   cd tryhackmeApp
   yarn / npm i
   ```

2. Start the app

   ```bash
   npx expo start

   npm run ios
   ```

## Features:

Onboarding

Mocked Login

Local quiz (no api / offline first)

Quiz contains 8 question, mixture of true / false,
Multiple choice (single answer), Open text input

Progress indicator

Ability to pass / fail the quiz

Quiz complete screen

Wrong answer triggers shake animation

Basic unit tests for core components
